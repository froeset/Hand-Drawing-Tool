# Hand Gesture Drawing

A real-time hand gesture drawing application using **MediaPipe Hand Landmarker** and **OpenCV**.  
Draw on a virtual canvas using your pointer finger, and move the canvas by pinching.  

## Features

- Draw with your **index finger**.
- **Pinch to pan** the canvas.
- Detect **fist, thumbs up, middle finger, pointer up**, and more.
- Real-time visualization with webcam feed.
- Configurable canvas size for more drawing space.


## Requirements

- Python 3.10+
- OpenCV
- MediaPipe Tasks
